# delete

Удаляет существующую комнату. Удалить может только создатель или администратор.

##HTTP METHODS
* `POST`

###Идентификатор метода
`41`

## Параметры

###Обязательные
    
* roomId - ```unsigned int``` 

    Идентификатор комнаты.
    
* userId - ```unsigned int``` 
    
    Идентификатор создателя.

Пример запроса:
```json
{
    "method":"create",
    "params": {
        "userId":123,
        "roomId":100       
    }
}
```

Пример ответа:
```json
{
    "code":"0",
    "response": {
        "roomId":100
    }
}
```
