# info
Возвращает информацию о комнате (пользователей, количество мест в комнате).<br>
По умолчанию возвращает
- количество пользователей в комнате,
- количество готовых к игре пользователей,
- количество мест,
- создателя.
- время игры.

##HTTP METHODS
* `GET`

###Идентификатор метода
`42`

## Параметры

###Обязательные
    
* roomId - целое беззнаковое. 

    Идентификатор комнаты.
    
###Опциональные
* users - boolean.

    Если `true` возвращается массив с пользователями, находящимися в комнате.

Пример запроса:
```json
{
    "method":"info",
    "params": {
        "roomId":100,
        "users":true
    }
}
```

Пример ответа:
```json
{
    "code":"0",    
    "response": {
        "roomId":100,
        "max_count":6,
        "users_count":3,
        "ready_users":2,
        "users":[
            {   "name":"Dmitry",

                "ready":true
                }, 
            {   "name":"Vasya", 
                "last_name":"Pupkin",
                "ready":true
                }, 
            {   "name":"Peter",
                "last_name":"Zaitsev",
                }

    }
}
```
