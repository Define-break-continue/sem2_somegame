# changeStatus

Меняет статус пользователя в комнате.

##HTTP METHODS
* `POST`

###Идентификатор метода
`44`

## Параметры

###Обязательные
    
* `userId` - целое беззнаковое. 
* `roomId` - целое беззнаковое. 
* `status` - unsigned byte.
    Значения:
        * `0` - не готов к игре
        * `1` - готов к игре
        * `2` - играет

Пример запроса:
```json
{
    "method":"join",
    "params": {
        "userId":123,
        "roomId":100,
        "status":1
    }
}
```

Пример ответа:
```json
{
    "code":"0",
    "response": {
        "userId":123,
        "roomId":100,
        "status":1
    }
}
```

Как только в комнате все игроки готовы, на фронт отправляется запрос с таймером о старте игры.
