# start

Запускает игру. <br>
После выполнения запроса каждому пользователю в комнате создается `gamerId` от 0 до 9.

##HTTP METHODS
* `POST`

###Идентификатор метода
`33`

## Параметры

###Обязательные
    * `roomId` - целое беззнаковое.  

Пример запроса:
```json
{
    "method":"start",
    "params": {
        "roomId":100       
    }
}
```

Пример ответа:
```json
{
    "code":"0",
    "response": {
        "roomId":100, 
        "gamers":[{
                    "userId":123,
                    "gamerId":0
                  },
                  {
                    "userId":124,
                    "gamerId":1
                  }                 
        ]              
    }
}
```
